This repository contains submissions for the "Algorithmic Toolbox" course by UC San Diego.

The course covers the design and efficient implementation of various algorithms for real-world scenarios. Assessments showcase the various situations one may encounter, and teach methods on how to identify 
constrains, as well as optimal solutions for problems dealing with: sorting and searching, divide and conquer, greedy algorithms, and more with a focus on dynamic programming. 

Folders (alongside submissions) are organized per week, with each expanding on the previous week's lesson. A broad outline of the course is listed as follows:

Week 1:
> Introduction

Week 2:
> Runtime estimation\
> Algorithm efficiency

Week 3:
> Greedy algorithms and maximization problems

Week 4:
> Divide and conquer algorithms

Week 5:
> Dynamic Programming

Week 6:
> Improving efficiency for dynamic programming
